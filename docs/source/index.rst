Welcome to the documentation of AFMD's sEQE setup !
===========================================================================================

Software to control sensitive external quantum efficiency (sEQE) setup of the Advanced Functional Materials and Devices (AFMD) Group. The setup is currently running on Linux but is being made compatible with Windows. Mac compatibility will be tested in the future.
